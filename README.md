## Apa itu Klipix? <a id="fitur"></a>

Buat yang pernah punya 20 ide di notes, 3 draft di Google Docs, dan 0 video yang kepublish — **Klipix** solusinya.

Klipix adalah workspace buat short-form creator (TikTok, Reels, YouTube Shorts) yang mau ngatur konten dari ide sampe publish, semua di satu tempat. Ditambah AI yang siap bantu brainstorming, nulis script, dan nyusun scene.

## Fitur

### 🧠 AI Generator
**Ide, script, dan scene — otomatis.**  
Tinggal masukin topik, pilih platform & tone, AI langsung kasih 8 ide konten + script lengkap + breakdown scene. Udah gitu support berbagai bahasa.

### 📥 Idea Inbox
**Nangkep ide secepat kilat.**  
Tulis ide, simpen ref link, dan promosiin ide terbaik jadi project beneran. Gak ada ide yang lenyap lagi.

### 📋 Script & Scene Pipeline
**Dari naskah ke scene, terstruktur.**  
Tulis master script, terus pecah jadi scene-scene — masing-masing punya script sendiri, durasi, dan status completion.

### 📊 Kanban Board
**Pipeline visual yang bisa di-drag.**  
Seret project dari Idea → Scripting → Shooting → Editing → Posted. Tau persis posisi tiap konten.

### 📱 Multi-Platform
**Satu pipeline, semua platform.**  
Label project pake TikTok, Instagram Reels, YouTube Shorts — atau semuanya.

### 🎨 Drag & Drop Scene Editor
**Atur urutan scene tinggal seret.**  
Rapi, intuitif, gak pake ribet.

### ⏰ Deadline Tracking
**Gak bakal telat publish lagi.**  
Set deadline, liat yang due soon langsung dari dashboard.

### 🌗 Dark & Light Mode
**Mata gak capek.**  
Mode gelap dan terang, tinggal klik.

## Kenapa Klipix? <a id="tanya-kenapa"></a>

Bukan cuma project manager biasa. Klipix dibikin khusus buat **short-form creator** yang butuh:

- **Cepet** — capture ide dalam hitungan detik  
- **Visual** — drag & drop, pipeline, progress bars  
- **AI-powered** — generator dari ide sampe script  
- **Fun** — neo-brutalist design, doodle icons, playful shadow  
- **Multi-platform** — semua konten di satu tempat  

## Mulai <a id="mulai"></a>

```bash
npm install
npm run dev
```

Butuh **Supabase** buat database & auth, dan **OpenRouter API key** buat fitur AI.

---

**Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS v4, Supabase, OpenRouter (Gemini).
